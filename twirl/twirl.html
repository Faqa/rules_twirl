Bazel rules for running the Twirl Template Compiler on Twirl Template Files

twirl_templates

twirl_templates(name, srcs, additional_imports, include_play_imports, source_directory, template_formats)

Compiles Twirl templates to Scala sources files.

Attributes

name

Name; Required

A unique name for this rule.
srcs

List of labels; Required

The actual template files contained in the source_directories.
additional_imports

List of strings; Optional; Default is []

Additional imports to import to the Twirl templates.
include_play_imports

Boolean; Optional; Default is False

If true, include the imports the Play project includes by default.
source_directory

Label; Required

Directories where Twirl template files will be found.
template_formats

Dictionary mapping strings to string; Optional; Default is {}

Formatter types for file extensions.

The default formats are

"html" -&gt; "play.twirl.api.HtmlFormat",
"txt" -&gt; "play.twirl.api.TxtFormat",
"xml" -&gt; "play.twirl.api.XmlFormat",
"js" -&gt; "play.twirl.api.JavaScriptFormat"
